# Display-a-4-digit-7-segment-LED-module-for-STM32F103C8T6
// Module LED Common-Cathode 5641AS
// Configuration
1. HCLK f = 72MHz
2. Timer 2:
  - Prescaler: 72-1
  - Counter Period: 0xffff-1
3. GPIO: Output
  * a - PC13
  * b - PA0
  * c - PA1
  * d - PA2
  * e - PA3
  * f - PA4
  * g - PA5
  * D1 - PA6
  * D2 - PA7
  * D3 - PB0
  * D4 - PB1
